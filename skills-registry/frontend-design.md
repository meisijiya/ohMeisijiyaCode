# 🎨 Frontend — UI/UX Design

> **使用**: `npx skills add <repo> --skill <name> -a opencode -g`

Skills to help AI generate polished, production-grade user interfaces with distinctive visual identity (not "AI-looking" designs).

---

## nextlevelbuilder — UI-UX-Pro-Max

**Source**: [`nextlevelbuilder/ui-ux-pro-max-skill`](https://github.com/nextlevelbuilder/ui-ux-pro-max-skill)

```bash
npx skills add nextlevelbuilder/ui-ux-pro-max-skill -a opencode -g -y
```

Purpose: Professional UI/UX design skill covering 50+ design styles, 21 color schemes, 50+ font pairings, 20+ chart types, and 8 tech stacks. Helps AI generate interfaces that look human-designed, not AI-generated.

## Anthropic — frontend-design

**Source**: [`anthropics/skills`](https://github.com/anthropics/skills)

```bash
npx skills add anthropics/skills --skill frontend-design -a opencode -g
```

## Tailwind Design System Support

Both `frontend-design` (Anthropic) and `web-design-guidelines` (Vercel Labs) include:

- Token-based color & spacing systems
- Component variant patterns
- Responsive utility class organization
- Dark mode implementation strategies
- Consistent animation patterns

---

## Why These Complement Our System

Our 11 global skills handle **process/workflow** (plan, implement, test, debug). These design skills handle **aesthetic judgment** — something our process skills intentionally avoid. They teach AI to make opinionated design choices rather than defaulting to bland "AI-style" output.
